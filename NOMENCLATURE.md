# NOMENCLATURE.md - Identifiers & Technical Data Reference

This document defines the common naming conventions and identifiers used across Pasmara Srl systems (PrestaShop, Kanguro, Wallaby).

## Product Identifiers

### SKU (`reference`)
- **Definition:** The unique alphanumeric code used to identify a specific product or variant in the warehouse and across systems.
- **Source of Truth:** Kanguro (ERP) and Wallaby (Catalog).
- **Format:** Usually strings like `SKU-12345` or `BASE-SKU`.
- **Note:** In Kanguro, the SKU is the primary identifier for a physical item.

### PrestaShop Product ID (`id_product`)
- **Definition:** The numeric ID assigned by PrestaShop to a base product (the parent entity).
- **Usage:** Identifies the general product page on the website.
- **Kanguro Mapping:** Stored in the field `external_reference`.

### PrestaShop Attribute ID (`id_product_attribute`)
- **Definition:** The numeric ID assigned by PrestaShop to a specific variant (combination) of a product (e.g., Color: Red, Size: XL).
- **Usage:** Identifies the exact variant chosen by the customer.
- **Kanguro Mapping:** Stored in the field `external_attribute_reference`.

## Order Identifiers

### Order ID (`id_order`)
- **Definition:** The internal numeric ID generated by PrestaShop when an order is placed.
- **Usage:** Primary key for order-related tables in PrestaShop.

### Order Reference (`reference`)
- **Definition:** A human-readable alphanumeric string (e.g., `ABCDEFGHI`) generated by PrestaShop.
- **Usage:** Shown to customers and often used in communications.
- **Note:** In Kanguro, this is stored as the order reference/number.

## System-Specific Mapping

| Term | PrestaShop Field | Kanguro Field | Wallaby Field |
| :--- | :--- | :--- | :--- |
| **Product ID** | `id_product` | `external_reference` | `id_product` |
| **Variant ID** | `id_product_attribute` | `external_attribute_reference` | `id_product_attribute` |
| **SKU** | `reference` | `reference` | `reference` |
| **Order ID** | `id_order` | `external_id_order` | N/A |
| **Order Ref** | `reference` | `order_number` | N/A |
